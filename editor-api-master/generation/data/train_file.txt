348	议题界定和决策事项确定。国企央企的决策性会议一般包括总经理办公会、党委会、董事会、股东会4种	一些具体的事还可以通过绩效委员会、招标委员会、安全委员会等专门委员会决定。有些议题该上什么会、上会需要领导决策什么、需要走哪些流程，职能部门甚至分管领导弄不清楚的时候，就是党委秘书来提建议了。党委秘书必须熟知本单位的党委议事规则、总经理办公会议事规则、公司章程、“三重一大”制度，以及各专门委员会的职责分工，结合决策事项的具体情况做出判断。比如，要不要成立某机构的问题得上党委会，如果党委同意成立这个机构，明确了职责分工，下一步由经营层来决定职数、人员调整（干部除外）等事宜就行了。再比如，年终奖金发放的原则，党委已经进行了安排“按照XX原则发放专项奖A、按照经营指标发放绩效奖B、向XXX项目、XXX项目、XXX项目发放专项奖励C，剩余由绩效委员会按照D原则发放”，那么后续发放的具体数字，如果没有灵活操作的余地，只需要绩效委员会把D的数额定一下，就可以发了，不用再过党委会了。这是既保障了“三重一大”事项依法依规决策，又压减了党委会议次数，不至于大事小事都上党委会。
